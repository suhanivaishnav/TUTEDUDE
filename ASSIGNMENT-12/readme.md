## Objective
To automate a browser using Selenium WebDriver and extract data from a website using Python.

## Description
This project uses Selenium WebDriver to open a website and automatically fetch data.
The script navigates to the website **https://quotes.toscrape.com/** and extracts:
- Quote text
- Author name

The extracted data is displayed in the console.

## Tools & Technologies
- Python 3
- Selenium WebDriver
- Google Chrome
- ChromeDriver

- `get_data.py` : Python script to automate browser and extract data
