# Chat Application using Socket Programming

## Module
Module 26: Network Programming in Python Using Socket

## Objective
To implement a client-server chat application using Python socket programming.

## Description
This project demonstrates basic network communication using TCP sockets.
A server listens for client connections and allows two-way communication
between the client and the server using the terminal.

## Files in Project
- server.py : Server-side chat program
- client.py : Client-side chat program
- README.md : Project documentation
- requirements.txt : Dependencies (none)

## Tools & Technologies
- Python 3
- Socket module
- TCP/IP protocol

## How to Run the Project
