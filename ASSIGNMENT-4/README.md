# Python Assignment – File Handling (Read, Write, Append)

This repository contains Python programs demonstrating basic file handling tasks.

---

## ✔ Task 1: Read a File and Handle Errors
### Description:
- Attempts to open and read `sample.txt`.
- Prints each line from the file.
- Handles the case where the file does not exist using try-except.

### File:
`task1.py`

---

## ✔ Task 2: Write and Append Data to a File
### Description:
- Takes user input and writes it to `output.txt`.
- Appends additional text to the same file.
- Reads and prints the final file content.

### File:
`task2.py`

---

## ✔ How to Run

Use these commands:

python task1.py
python task2.py
