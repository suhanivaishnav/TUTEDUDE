# Python Assignment – Dictionary & List Slicing

This repository contains two Python programs based on dictionary operations and list slicing.

---

## ✔ Task 1: Dictionary of Student Marks
### Description:
- Creates a dictionary with student names as keys and marks as values.
- Takes user input to search for a student's marks.
- Displays the marks if found.
- Shows an appropriate message if the student does not exist.

### File:
`task1.py`

---

## ✔ Task 2: Demonstrate List Slicing
### Description:
- Creates a list of numbers from 1 to 10.
- Extracts the first five elements using slicing.
- Reverses the extracted list.
- Prints both lists.

### File:
`task2.py`

---

## ✔ How to Run the Programs

Use the commands:
A-5 task-1.py
A-5 task-2.py
